# Python POO

## O que faz: 
Cria uma classe Funcionario que armazena nome, idade, cargo e salário.

## Funcionalidades: 

- Cálculo de aumento salarial percentual.
- Alteração de cargo.
- Listagem formatada de dados do funcionário.

## Conceitos Aplicados: 

Classes, Métodos, Atributos de Instância e de Classe, e F-strings para formatação de moeda (R$).
